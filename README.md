# Steps to Use

## 1. Fork with All Branches

## 2. Clone Repository

```bash
git clone <REPO_LINK>
```

## 3. Navigate to Project Directory

```bash
cd <REPO_NAME>
```

## 4. Pull Required Branch

```bash
git pull origin <REQUIRED_BRANCH_NAME>
```

## 5. Install pnpm if not installed

```bash
npm i -g pnpm
```

## 6. Install Dependencies

```bash
pnpm install
```
